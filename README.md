### How to Run This Project ###

* run backend server
```
cd backend
```
install vendor
```
composer update
```
* migration 
```
php artisan migrate
```
##### Or
imnport the sql file on the database
* running server

```
php artisan serve
```
```
cd ..
```

#### run frontend ###
```
cd frontend
```
* installing node module

```
yarn
``` 
or

``` 
npm i
```
#
* start frontend server
```
yarn start
```

Goto http://127.0.0.1:3000 (default)
